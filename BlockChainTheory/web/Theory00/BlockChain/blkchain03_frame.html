Block Chain


How Alice can send money using the block chain

  • 1 Alice sends the transaction to it's node
  • 2 The node validates the transaction APPLY(Status,Transaction) -> Status'. If valid transaction added to queue of pending transactins
  • 3 The node sends the transaction to the other nodes
  • 4 Each nodes validates the transaction and add it to it's own queue of pending transations
