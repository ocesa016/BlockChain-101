Block Chain


How Alice can send money using the block chain

  • 1 Alice sends the transaction to its node
  • 2 The node validates the transaction APPLY(Status,Transaction) -> Status'. If valid, transaction is added to the queue of pending transactions
  • 3 The node sends the transaction to the other nodes
  • 4 Each node validates the transaction and adds it to its own queue of pending transations
