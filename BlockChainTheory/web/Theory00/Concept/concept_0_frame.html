Block Chain Concepts

Elements

  • Transactions. Transactions are created by participants using the system.
  • Blocks. Blocks record and confirm when and in what sequence transactions enter the block chain..
Building Blocks

  • Partecipants. Block Chain users that interact with each other by mean of the Block Chain Network.
  • A network of peer-to-peer nodes. A blockchain network can include everyone with a computer or a small group of known entities that agree to participate.
  • A consensus mechanism. The set of rules the network uses to verify each transaction and agree on the current state of the blockchain.
