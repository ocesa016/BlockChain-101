Liquid

Liquid is the First Sidechain for Bitcoin Exchanges

Announced in October 2015 by BlockStram [ Announce ]

Aim is to improve capital efficiency and market liquidity by facilitating rapid and secure transfers between accounts held at any participating exchange or brokerage.

Blockstream, raised $21 million in venture capital late last year to deploy Liquid to be used among exchanges, wallets and other Bitcoin service providers.
It is unclear if Liquid was ever put it into production. Point is that Liquid is a private sidechain that doesn't show statistics. It's based on open source code that is being adopted for the main BitCoin Blockchain at the moment.
